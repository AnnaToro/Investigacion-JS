1- ¿Qué versiones de JavaScript hay en el mercado?.

En el mercado se encuentran 8 versones de JavaScript, estas son:

  • ES6: Se creo una opcion para declarar variables, estas fueron nombradas como:Let (permitiendo declarar variables limitando su alcance), Const ( Declara solo variable del tipo lectura.) y Arrow (Esta es limitada y no se puede utilizar en todas las situaciones.)
  • ES7: Se introdujo una alternativa a Math.pow para poder realizar operaciones más claras:
  • ES8: Fue diseñada la funcion Object entries, esta nos permite transformar un objeto en un array, con cada una de las propiedades de un array de la forma [key,value];
  • ES9: Se incorporó la posibilidad de usar grupos nombrados usando una expresión regular y tambien un match de una expresión regular retorna un objeto.
  • ES10: Fue incorporado el método flat de los array permite aplanar los elementos internos de un array permitiendo eliminar array internos.
  • ES11: Fue creada el "Optional chaining", es una forma de evitar errores solicitando propiedades de forma opcional utilizando el símbolo ? , sirve cuando no estamos seguros de que los valores estén completos.
  • ES12: En esta version se diseño "Numeric-separators", permitiendonos separar los números con _, con la finalidad que la lectura del número sea más fácil y clara.
  • ES13: Y por ultimo, en la version mas reciente, fue diseñado el metodo "At", este recibe un valor numérico entero y devuelve el elemento en esa posición, permitiendo valores positivos y negativos. Los valores negativos contarán desde el último elemento del array.
