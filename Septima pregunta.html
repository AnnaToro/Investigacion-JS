7- ¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?.

Cuando hablamos de objeto DOM, estamos refiriendonos a la interfaz de programación que nos permite crear, cambiar, o remover elementos del documento. El dom trabaja de 3 formas:

  • getElementById(): Este no permite tener el mismo nombre de id para dos elementos diferentes.
  • querySelector(): Puedes usar este método para encontrar elementos con uno o más selectores de CSS.
  • querySelectorAll(): Este método encuentra todos los elementos que coinciden con el selector de CSS y devuelve una lista de todos esos nodos.
