2- ¿Cómo se integra JavaScript a HTML?.

Existen dos formas de incluir JavaScript a HTML; La primera forma es utilizando la etiqueta script , esta va dentro de la etiqueta head y body. La segunda forma es por externa, en esta forma tambien se utiliza la etiqueta script , pero a diferencia de la otra, se utiliza el atributo src, con la finalidad de poder especificar la URL del archivo JavaScript
